# MounjaroPLAN
마운자로 맞는 플랜

## 웹 실행 방법
간단한 Flask 서버로 웹에서 리포트를 생성할 수 있습니다.

1. 의존성 설치
   ```bash
   pip install -r requirements.txt
   ```
2. 개발 서버 실행
   ```bash
   python app.py
   ```
3. 브라우저에서 `http://localhost:8000` 에 접속 후 입력 폼을 채워 리포트를 생성하세요. JSON API가 필요하면 `/api/report` 엔드포인트에 POST 요청을 보내면 됩니다.
